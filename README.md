ChromaPulse: Non-Contact Health Monitoring
Computer Vision Capstone Project (BYOP)

Overview
ChromaPulse uses Remote Photoplethysmography (rPPG) to detect tiny, invisible-to-the-eye color changes in human skin to estimate heart rate. It uses MediaPipe Face Mesh to track facial landmarks in real-time, ensuring the "sensor" stays locked on the forehead even if the user moves.

Tech Stack
Language: Python 3.10+

CV Library: OpenCV & MediaPipe

Signal Processing: NumPy & SciPy

How to Run
Clone the repo.

Install dependencies: pip install -r requirements.txt

Run the app: python app.py
